# Description

A Drupal 6.x feature that demonstrates some of the basic features of OpenLayers

# Requirements

A Drupal 6.x installation.

Drupal modules:
* ctools
* content
* views
* openlayers, openlayers_cck, openlayers_views
* openlayers_geocoder
* features

# Installation

* Enable this module.
* Add a story with a location
* Check out /stories
